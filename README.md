# Flight Docs

CLI access to Alces Flight Center documents

## Overview

This is a command line utility to provide access to your Alces Flight Center
documents.

## Installation

```
git clone https://github.com/alces-flight/flight-docs.git
cd flight-docs
bundle install
```

## Operation

The tool can be executed using the `bin/docs` script.
